# 需求文档目录

本目录把 RepoPilot 的项目愿景拆成可开发、可验收的需求规格。

## 文件说明

| 文件 | 内容 |
| --- | --- |
| [product-requirements.md](./product-requirements.md) | 项目背景、目标用户、业务目标、功能需求和非功能需求 |
| [mvp-scope.md](./mvp-scope.md) | MVP 必做范围、暂缓范围、阶段边界和验收标准 |
| [user-stories.md](./user-stories.md) | 按角色拆分用户故事和验收条件 |

## 需求编号规则

- `PRD-F-*`：功能需求。
- `PRD-NF-*`：非功能需求。
- `MVP-*`：MVP 范围与验收项。
- `US-*`：用户故事。
